## Notes

- This document is maintained and updated as new UNESCO sites are added to the application.
- Site counts are based on the CSV files in the `public/data` directory.
- Transnational sites (shared by multiple countries) are listed under each participating country.
- Some countries may have sites that are part of larger transnational nominations (e.g., Silk Roads corridors, Western Tien-Shan).

## How to Update This Document

To update the site counts in this document:

1. **Automatic Count (Recommended):**
   ```bash
   # Count sites in all UNESCO CSV files
   find public/data -name "*_UNESCO_Sites.csv" -exec sh -c 'country=$(basename "$1" _UNESCO_Sites.csv | tr "_" " "); count=$(tail -n +2 "$1" 2>/dev/null | grep -c . || echo 0); echo "$country|$count"' _ {} \; | sort
   ```

2. **Manual Update:**
   - Open the CSV files in `public/data` directory
   - Count the data rows (excluding header)
   - Update the corresponding sections in this document

3. **When Adding New Countries:**
   - Add the country to the appropriate regional section
   - List all UNESCO sites with their inscription year
   - Update the statistics summary at the bottom
   - Add an entry to the Update History section
